# Recharge Plus - VTU Website

Recharge Plus is a Virtual Top-Up (VTU) platform that allows users to purchase airtime, data bundles, electricity, TV subscriptions, and convert airtime to cash. The platform includes separate user and admin areas for a seamless experience.

## Features

### User Area
- **Landing Page**: Welcome page with platform overview, features, and services.
- **User Authentication**: Login and registration for users.
- **User Dashboard**: Overview of account balance, recent transactions, and quick actions.
- **Purchase Sections**:
  - Data: Buy data bundles for various networks.
  - Airtime: Recharge airtime for mobile networks.
  - Electricity: Pay electricity bills.
  - TV Subscription: Subscribe to cable TV services.
  - Airtime to Cash: Convert airtime to cash.

### Admin Area
- **Services Management**: Admin can add and manage services (e.g., networks, providers).
- **Plans Management**: Admin can add plans with:
  - Plan ID
  - Plan Duration
  - Price of the Plan
- **Direct Pricing**: Admin can set custom prices for each plan.
